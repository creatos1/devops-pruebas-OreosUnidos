# Comandos para ejecutar las 3 pruebas en Visual Studio / PowerShell

## 1️⃣ Levantar la API

En una terminal separada:

python -m uvicorn src.main:app --host 0.0.0.0 --port 8000

---

## 2️⃣ Pruebas unitarias

En otra terminal:

python -m pytest tests/test_api.py --cov=src

---

## 3️⃣ Pruebas de integración

Si los tests de integración están en el mismo archivo `tests/test_api.py`:

python -m pytest tests/test_api.py --cov=src

> Alternativamente, si quieres separarlos:
> python -m pytest tests/test_integration.py --cov=src

---

## 4️⃣ Pruebas funcionales (Postman / Newman)

En otra terminal:

newman run tests/api/test_api.postman_collection.json --env-var "FASTAPI_URL=[http://localhost:8000](http://localhost:8000)"
